- Microsoft is releasing tools that clean up PCs harbouring viruses and spyware.
- A software utility will help users find and remove any spyware on their home computer.
- SP2 upgrade for Windows XP closed many security loopholes.
- 90% of Windows machines have the malicious software on board.
- Each one harbours 28 separate spyware programs on average.
- The virus-fighting program will be updated monthly.
- It is a precursor to Microsoft releasing dedicated anti-virus software.
- Before now Microsoft has left the market for PC security software to specialist firms.
- In 2003 it bought Romanian firm GeCAD Software.
- In December 2004 it bought New York-based anti-spyware firm Giant Company Software.
